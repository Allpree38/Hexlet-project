# «Генератор резюме»

## Цель проекта: разработать интерактивную информационную систему для создания
профессиональных резюме в несколько шагов. Система должна упростить процесс
составления резюме для соискателей, предлагая структурированное заполнение данных,
готовые шаблоны и рекомендации по улучшению текста.

## Целевая аудитория: соискатели работы, студенты, специалисты, желающие обновить
своё резюме.

## Технология
- CSS
- HTML
- JavaScript
